# Modules Library

This folder stores dashboard-ready training resources for the ZETRC learning library.

Structure:
- `pdf/english`: English PDF modules
- `pdf/nyanja`: Nyanja PDF modules
- `audio/english`: English audio lessons
- `audio/nyanja`: Nyanja audio lessons
- `video/english`: English video uploads
- `video/nyanja`: Nyanja video uploads

Starter content has been added so the dashboard can display the library immediately.
Replace the placeholder files with your final lesson assets when they are ready.
